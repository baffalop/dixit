# Dixit

A realtime online version of the Dixit boardgame. Made using WebSockets, Node (TypeScript, Express), and Vue (TypeScript).

### Project setup
```
npm install
```

### Run development servers
Hot-reload Vue server
```
npm run vue
```

Node API/WebSocket server (proxied by Vue)
```
npm run serve
```

### Build
```
npm run buildall
```

### Lint and fix files
```
npm run lint
```
